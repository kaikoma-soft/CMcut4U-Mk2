
・使用する環境変数の名前と意味

  FFMPEG   : ffmpeg の実行ファイル
  INPUT    : 入力ファイル名
  OUTPUT   : 出ファイル名
  SS       : 開始時間
  WIDTH    : 経過時間
  VFOPT    : -vf オプション
  SIZE     : 画面サイズ
  WAVRATIO : wav のサンプリングレシオ
  LOGLEVEL : ログレベル fatal
  FRAMES   : フレーム数
  H265PRESET: H265 PRESET
  MONO     : ２カ国語対応のオプション
  METAFN   : メタデータのファイル名
